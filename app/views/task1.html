Task description:

Написати сценарій на JS який згенерує і виведе в браузер таблицю Піфагора, приблизний вигляд якої зображений на рисунку 2.1

Особливості:
  1. Існує головний контейнер в який будуть додані відповідні контейнериз числами.
  2. Головний контейнер робиться статично засобами HTML і стилізується засобами CSS. Для цілого документа потрібно використати моноширний шрифт (літери однакової ширини) для прикладу Courier.
  3. Контейнери з числами генеруються засобами JavaScript з використанням DOM-методів: createElement(), setAttribute(), appendChild(), getElementById, об’єкту document.
  4. За бажанням можна стилізувати контейнери з числами у відповідності до числа (парне/непарне, або інші особливості).
